# v0.1.139 - 评论弹窗避让播放器与液态玻璃

## 更新内容

- 评论弹窗高度按当前播放器高度和安全区动态计算，弹窗只占用视频窗口下方区域，避免覆盖播放器画面。
- 评论弹窗使用透明系统 sheet 背景，并将评论面板改为 iOS 26 Liquid Glass 风格。
- “查看评论”入口按钮、评论排序胶囊、评论输入占位框同步接入液态玻璃交互效果。
- iOS 26 以下系统保留 `.ultraThinMaterial` 毛玻璃兜底，避免旧系统显示异常。
- 评论弹窗内容改为独立滚动，低高度状态下仍可正常浏览评论和加载更多。

## 验证建议

- 竖屏进入视频详情页，点击“查看评论”，确认弹窗顶部不会压住视频窗口。
- 旋转或不同尺寸设备下打开评论，确认弹窗高度会跟随可用区域调整。
- 在 iOS 26 上确认评论入口、排序、输入框和评论面板呈现液态玻璃质感。
- 在低版本 iOS 上确认毛玻璃兜底正常，评论可滚动、刷新、加载更多。

## 打包说明

- GitHub Actions 会构建 Release 配置的未签名 IPA。
- 生成的 `ccbili-unsigned.ipa` 会作为 Release 附件上传。
- 未签名 IPA 需要自行重签名后安装到真机。
