# v0.1.143 - 评论弹窗避让播放器与内容区右滑返回

## 更新内容

- 评论弹窗高度改为按视频播放器的实际下边缘动态计算，弹窗顶部不会超过播放器下边缘。
- 移除评论弹窗高度的固定最小值兜底，避免小屏或特殊比例下弹窗反向压住视频窗口。
- 视频窗口之外的页面区域新增任意位置右滑返回，评论弹窗未打开时可直接从详情内容区右滑回上一页。
- 右滑返回会避开视频窗口区域，避免和播放器手势、系统控制条冲突。

## 验证建议

- 打开视频详情页后点击“查看评论”，确认评论弹窗顶部始终位于视频播放器下边缘以下。
- 在视频窗口之外的信息卡、评论按钮、简介区域横向右滑，确认能返回上一页。
- 在视频窗口内部右滑，确认不会误触发页面返回。
- 评论弹窗打开时右滑详情背景，确认不会触发页面返回。

## 打包说明

- GitHub Actions 会构建 Release 配置的未签名 IPA。
- 生成的 `ccbili-unsigned.ipa` 会作为 Release 附件上传。
- 未签名 IPA 需要自行重签名后安装到真机。
