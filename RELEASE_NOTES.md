# v0.1.147 - 详情页原生透明液态玻璃微调

## 更新内容

- 视频详情页三个主容器移除额外白色 tint，改回 iOS 26 原生 `.regular` Liquid Glass 透明质感。
- 评论入口保留 `.interactive()` 触感，但不再额外增白，和信息卡、简介推荐保持同一透明层级。
- 保留上一版评论弹窗、回复跳转修复、用户主页刷新移除、滑动性能和竖屏播放器控件修复。

## 验证建议

- 在 iOS 26 真机打开视频详情页，确认信息卡、评论入口、简介推荐都更透明，更接近系统原生 Liquid Glass。
- 在亮色和暗色背景视频下分别检查，确认文字仍然清晰可读。
- 点击“查看评论”和竖屏播放器，确认评论弹窗、回复弹窗、原生播放器控件仍正常。

## 打包说明

- GitHub Actions 会构建 Release 配置的未签名 IPA。
- 生成的 `ccbili-unsigned.ipa` 会作为 Release 附件上传。
- 未签名 IPA 需要自行重签名后安装到真机。
