# v0.1.137 - 视频详情操作区对齐修复

## 更新内容

- 修复视频详情卡片内标题、信息行与底部操作区视觉边距不一致的问题。
- 底部收藏、点赞、投币、分享改为统一的一整条系统分组背景容器。
- 移除每个操作按钮单独的背景块，保留原有点击范围、加载状态和交互逻辑。
- 操作区左右边界现在与标题和清晰度信息行保持同一套卡片内边距。

## 验证建议

- 打开视频详情页，确认标题左边缘、清晰度行和底部操作区外边界视觉一致。
- 点击收藏、点赞、投币、分享，确认功能仍可正常响应。
- 切换深色/浅色模式，确认操作区背景符合 iOS 分组卡片层级。

## 打包说明

- GitHub Actions 会构建 Release 配置的未签名 IPA。
- 生成的 `ccbili-unsigned.ipa` 会作为 Release 附件上传。
- 未签名 IPA 需要自行重签名后安装到真机。
