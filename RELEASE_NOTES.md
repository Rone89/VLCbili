# v0.1.149 - 详情页主容器改为纯 iOS 26 原生透明玻璃

## 更新内容

- 移除详情页三个主容器上的 `.regularMaterial` 底层和额外半透明填充，避免玻璃被糊厚。
- iOS 26 分支只保留原生 `.glassEffect(.regular)`，评论入口保留 `.interactive()`，透明度更接近系统原生 Liquid Glass。
- 详情页背景透明度轻微降低，让主容器玻璃能透出更多底层层次。
- 保留上一版评论弹窗、回复跳转修复、用户主页刷新移除、滑动性能和竖屏播放器控件修复。

## 验证建议

- 在 iOS 26 真机打开视频详情页，确认信息卡、评论入口、简介推荐明显更透明。
- 分别在亮色/暗色视频画面下检查文字可读性和玻璃边界。
- 点击“查看评论”和竖屏播放器，确认评论弹窗、回复弹窗、原生播放器控件仍正常。

## 打包说明

- GitHub Actions 会构建 Release 配置的未签名 IPA。
- 生成的 `ccbili-unsigned.ipa` 会作为 Release 附件上传。
- 未签名 IPA 需要自行重签名后安装到真机。
