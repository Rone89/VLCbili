# v0.1.152 - 修复竖屏原生播放器控制条

## 更新内容

- 修复视频详情页竖屏播放器点击后不显示 AVKit 原生进度条、暂停/播放按钮的问题。
- 将详情页结构从覆盖式层叠改为播放器固定在顶部、内容滚动区域位于播放器下方，避免透明滚动层拦截播放器触摸。
- 保留现有液态玻璃详情卡、评论入口和简介推荐布局，不改动播放器实例和播放链路。

## 验证建议

- 竖屏进入视频详情页，点击视频画面，确认系统原生进度条、播放/暂停按钮可正常显示。
- 点击进度条拖动、暂停/播放，确认手势能传给 AVKit 原生控制条。
- 上下滑动详情内容，确认内容从视频窗口下方开始，不遮挡视频窗口触摸。

## 打包说明

- GitHub Actions 会构建 Release 配置的未签名 IPA。
- 生成的 ccbili-unsigned.ipa 会作为 Release 附件上传。
- 未签名 IPA 需要自行重签名后安装到真机。
