# v0.1.144 - iOS 26 官方内容区返回与详情页交互清理

## 更新内容

- 右滑返回改为使用 iOS 26 官方 `interactiveContentPopGestureRecognizer`，支持内容区域中部右滑返回。
- 保留旧系统的 `interactivePopGestureRecognizer` 边缘返回兜底，不再使用自定义 SwiftUI DragGesture 模拟返回。
- 视频详情页播放器窗口固定为横向 16:9 显示比例，也就是高度:宽度 = 9:16。
- 删除视频详情页的下拉刷新功能，避免和返回/滚动手势冲突。
- 删除评论弹窗和评论回复弹窗内的下拉刷新功能，保留首次加载与切换排序加载。

## 验证建议

- 在 iOS 26 上从视频窗口外的内容区域中部右滑，确认由系统返回手势返回上一页。
- 在旧版 iOS 上从左边缘右滑，确认仍可返回上一页。
- 进入视频详情页，确认播放器窗口固定为 16:9，详情组件从播放器下方开始。
- 打开评论弹窗和回复弹窗，确认下拉不会触发刷新控件。

## 打包说明

- GitHub Actions 会构建 Release 配置的未签名 IPA。
- 生成的 `ccbili-unsigned.ipa` 会作为 Release 附件上传。
- 未签名 IPA 需要自行重签名后安装到真机。
