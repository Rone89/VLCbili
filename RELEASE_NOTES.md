# v0.1.150 - 修复视频详情标题左右边距观感

## 更新内容

- 修复视频详情页信息卡标题左右边距观感不一致：标题单独使用满宽布局，和下方标签、操作区保持同一内容宽度。
- 归一化视频标题中的换行和连续空白，避免标题第一行提前断开导致右侧出现大块留白。
- 保留上一版纯 iOS 26 原生透明 Liquid Glass、评论弹窗、回复跳转修复、滑动性能和竖屏播放器控件修复。

## 验证建议

- 打开长标题视频详情页，确认标题左边和右边都贴近信息卡统一内边距。
- 检查标题换行，确认不会因为原始标题中的换行或多余空白造成右侧异常空白。
- 检查清晰度标签、点赞收藏区和评论入口仍保持原有布局。

## 打包说明

- GitHub Actions 会构建 Release 配置的未签名 IPA。
- 生成的 `ccbili-unsigned.ipa` 会作为 Release 附件上传。
- 未签名 IPA 需要自行重签名后安装到真机。
